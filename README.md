## 🐝 디렉토리 구조

```
📂 web29-honeyflow/
├── 📂 packages/            모노레포의 패키지들이 위치
│   ├── 📂 backend/         백엔드 관련 패키지
│   │   ├── 📂 src/
│   │   ├── 📄 package.json
│   │   └── 📄 tsconfig.json
│   ├── 📂 frontend/        프론트엔드 관련 패키지
│   │   ├── 📂 src/
│   │   ├── 📄 package.json
│   │   └── 📄 tsconfig.json
│   └── 📂 shared/          프론트엔드와 백엔드에서 공용으로 사용하는 패키지

├── 📄 pnpm-lock.yaml
```
